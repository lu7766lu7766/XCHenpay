# 金流帳號

### API

> 獲取可用金流帳號(商戶角色用)

| 項目         | 內容                         | 類型         | 預設         | 說明                  | 必填  |
|-------------|-----------------------------|--------------|--------------|---------------------|-------|
| <b>路徑</b>  |user/payment/account/user/info        |              |              |                     |      |
| <b>方法</b>  | POST                        |              |              |                     |      |
| <b>權限</b>  |          |              |              |          -          |      |
| <b>參數</b>  | vendors   | array  |         |    金流代碼,不送顯示全部  |   x  |
